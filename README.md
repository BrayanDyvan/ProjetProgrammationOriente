Projet Synthèse POO - Gestion du Bunker 
1. Description générale
Application Java pour gérer les survivants d'un bunker après une invasion zombie. Utilise POO, fichiers, sérialisation et collections.
2. Fonctionnalités principales
Ajouter, supprimer, modifier, rechercher survivants; trier par date de naissance; exporter listes en fichiers texte; sauvegarde automatique bunker.ser.
3. Types de personnel
Forces de sécurité, Scientifiques, Administrateurs, Employés de maintenance (ingénieurs et ouvriers).
4. Options ajoutées
REGEX pour matricule, choix du nom et chemin des fichiers, interface ajoutée, enum pour recherche, attribut lieu de travail, tri avancé, Collections.shuffle et reverseOrder, validations try/catch, anti-doublon, exécutable fourni.
